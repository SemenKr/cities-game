# Cities Game

🚀 Интерактивная SPA-игра на React + TypeScript с реализованной бизнес-логикой, тестированием и оптимизацией производительности

👉 Демо: https://semenkr.github.io/cities-game/  
👉 Код: https://github.com/SemenKr/cities-game  

---

## ⚡ Ключевые возможности

- Полноценная игровая логика с проверкой правил и валидацией ввода  
- AI-соперник с несколькими уровнями сложности (easy / medium / hard)  
- Управление состоянием приложения и игровой сессией  
- Система подсказок и предотвращение повторов  
- Сохранение статистики в localStorage  
- Адаптивный UI (desktop / mobile)

---

## 🧠 Архитектура и подход

- Бизнес-логика вынесена в отдельные модули (`cityRules`, `gameStats`)  
- Покрытие ключевых сценариев unit- и интеграционными тестами  
- Разделение UI и логики для упрощения поддержки  
- Масштабируемая структура проекта  
- Реализован алгоритм выбора города для AI с учётом доступных вариантов и сложности
---

## ⚡ Производительность

- Lazy loading экранов  
- Code splitting (React, antd, rc-*)  
- Оптимизация бандла  

---

## 🛠 Стек

React 18 · TypeScript · Vite · Ant Design · SCSS Modules  
Vitest · Testing Library
